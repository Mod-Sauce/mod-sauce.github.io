OER (Otyacraft Engine Renewed)

OER, or Otyacraft Engine Renewed, is our project dedicated to updating the Otyacraft Engine to be compatible with the newest modloader versions for Minecraft.

About Otyacraft Engine

Otyacraft Engine is a core mod that provides essential functionality and API for other Otyacraft mods. It's a fundamental part of the Otyacraft mod ecosystem, but it required updates to work with modern modloader versions.

OER's Features

  • Modern Modloader Compatibility: Updated to work seamlessly with the latest versions of popular modloaders.
  • Stability Improvements: Addressed compatibility issues and bugs from the original engine.
  • Continued Support: We are committed to keeping OER up-to-date and reliable.
  • API Maintenance: Ensuring that the core API remains functional and usable for other Otyacraft mods.

Our Goal

Our goal with OER is to ensure that the Otyacraft mod ecosystem can continue to thrive on modern Minecraft versions. By updating the core engine, we are enabling other Otyacraft mods to remain relevant and enjoyable.
